# AutoSnake

Snake 2D (Python + pygame) construido con PRs incrementales.

## Estado actual
Bootstrap técnico listo:
- estructura de módulo `autosnake`
- loop con lógica por tick fijo + render desacoplado
- ventana pygame con salida por `ESC` o cierre de ventana

## Requisitos
- Python 3.10+

## Instalación
```bash
pip install -r requirements.txt
```

## Ejecución
```bash
PYTHONPATH=src python -m autosnake.main
```
