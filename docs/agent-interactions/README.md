# Agent Interactions Audit

Este directorio registra:
- qué agente fue invocado,
- con qué contexto,
- qué resultado devolvió,
- y oportunidades de mejora.

> Nota: por política de visibilidad, no siempre fue posible leer transcript completo de cada sesión subagente. Se registró todo lo observable desde la sesión orquestadora.

## Archivos
- `delegations.md` — prompts delegados a cada agente.
- `results.md` — resultados recibidos en la sesión principal.
- `improvements.md` — oportunidades de mejora del proceso multiagente.
